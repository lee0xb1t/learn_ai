本数据收集了各国的人均GDP数据的变化。
**数据描述**
本数据集是CSV格式的数据，包含了1800年到2040年中国、美国、英国、日本、俄国人均GDP的变化。后面的年份数据是通过预测模型预测出来的，本数据集可作为新手的练手项目。
